# Zed Demo Project

This is a full-stack demo project consisting of a Go backend and a Svelte frontend.

## Project Structure

- `backend/`: Contains the Go backend application.
- `frontend/`: Contains the Svelte frontend application.

## Getting Started

### Backend

To set up and run the backend:

1.  Navigate to the `backend` directory: `cd backend`
2.  Install dependencies (if any, typically `go mod tidy` takes care of this): `go mod download`
3.  Run the application: `go run cmd/main.go`

### Frontend

To set up and run the frontend:

1.  Navigate to the `frontend` directory: `cd frontend`
2.  Install dependencies: `npm install`
3.  Run the development server: `npm run dev`

## Further Information

Refer to the `backend/README.md` (if it existed) and `frontend/README.md` for more specific instructions on each part of the application.
